main_func (filepath, aname)
The main module functionality being executed. And another long description without any inspiration other than lots of coffee, and it was bad coffee, and so is the long description. Of special interest here is the 'example' tag which is being used with a trailing '#'. This will preserve whitespace and line-breaks, especially usefull to preserve the layout of example code. The '#' will also work on other tags. Other than that, the 'see' tag is automatically inferred because it references another, also documented, element.

Parameters:

  • filepath: The filepath to search for input files
  • aname: Filename of the output file

Usage:

preferably call this function once, directly after requiring it

Return value:

a table with all files, or nil + error

Example:

for _, filename in ipairs(filepath) do
    print("filename:", filename)
end
